# socket_using_c
socket programming using C
##### multi client version

### Requirements:
1) Windows OS
2) C version >= 99
3) cmake version >= 3.14
4) winsock2 and pthread libraries

### How to run:
#### Socket server:
1) cd socket_server
2) mkdir cmake
3) cd cmake
4) cmake ..
5) make
6) run socket_server binary file
#### Socket client:
1) cd socket_client
2) mkdir cmake
3) cd cmake
4) cmake ..
5) make
6) run socket_client binary file
##### You can test it now :) 
##### Also check Build dir for executables.
